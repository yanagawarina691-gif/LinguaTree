# LinguaTree Design Context

## Design Context

### Users
- 中文母语者，正在通过短视频学习英语知识点
- 使用场景：移动端、碎片时间、单次会话短
- 任务：查看已解析视频沉淀下来的知识点（矿石/水晶），追踪学习进度，点击进入复习

### Brand Personality
- 活泼、有成就感、游戏化成长
- 像一座可探索的知识矿洞，但本次页面要求“纯白底 + 晶体蛇形串联”
- 关键词：清澈、秩序、成长、收藏感

### Aesthetic Direction
- 纯白背景（#FFFFFF），高对比深色文字
- 彩色晶体作为内容主角，颜色保持宝石色系
- 蛇形/S 形路径：从上到下，左→右，下一行右→左，循环往复
- 始终有一条细腻的串联线贯穿所有晶体，暗示知识递进
- 避免深色矿洞、过多阴影、复杂渐变；保持轻盈通透

### Design Principles
1. 背景极简，让晶体颜色成为视觉焦点
2. 路径清晰，用户一眼看懂学习顺序
3. 触控友好，晶体间距足够，点击反馈明确
4. 信息克制，名称 + 等级即可，详情通过底部卡片展开
5. 保持 LinguaTree 原有游戏化语气（Lv.、XP、矿脉）
